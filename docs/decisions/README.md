# Decisions

이 디렉터리는 채택된 기술 결정만 ADR 형태로 보관한다.

운영 원칙은 다음과 같다.

- 아직 확정되지 않은 설계 비교, 전략 검토, 회의 메모는 [docs/discussions/](https://github.com/jho951/Block-server/blob/dev/docs/discussions/README.md)에 남긴다.
- 실제 채택된 결정만 이 디렉터리에 ADR로 추가한다.
- ADR이 특정 discussion 검토를 거쳐 채택되었다면, ADR에도 대응 discussion 링크를 함께 남긴다.
- [docs/REQUIREMENTS.md](https://github.com/jho951/Block-server/blob/dev/docs/REQUIREMENTS.md)는 현재 유효한 요구사항과 채택된 정책을 반영한다.
- [prompts/](https://github.com/jho951/Block-server/blob/dev/prompts/README.md)는 작업 로그와 초안 메모를 보관한다.

즉 문서 흐름은 다음 순서를 권장한다.

1. [docs/discussions/](https://github.com/jho951/Block-server/blob/dev/docs/discussions/README.md)에서 검토
2. [docs/decisions/](https://github.com/jho951/Block-server/blob/dev/docs/decisions/README.md)에서 채택 기록
3. [docs/REQUIREMENTS.md](https://github.com/jho951/Block-server/blob/dev/docs/REQUIREMENTS.md)에 최종 반영
