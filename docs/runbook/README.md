# Runbook 운영 메모

이 디렉토리는 장애, 오류, 설정 문제를 다시 재현하고 점검할 수 있는 절차를 보관한다.

## 무엇을 여기에 두는가

- 로컬/개발 환경에서의 재현 절차
- 장애 점검 순서
- 확인해야 할 로그, 쿼리, 엔드포인트
- 환경 설정 확인 항목
- 복구/우회 방법

## 작성 기준

- 가능한 한 다시 따라 하면 같은 문제를 재현하거나 같은 점검을 수행할 수 있게 적는다.
- 명령, 로그 위치, 선행 조건, 기대 결과를 같이 적는다.
- "안 되면 다시 해본다" 같은 모호한 표현보다 확인 포인트를 구체적으로 적는다.
- 실제 디버깅 흐름이 바뀌면 같은 작업에서 runbook도 갱신한다.
