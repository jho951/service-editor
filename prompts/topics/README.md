# Topics 운영 메모

이 디렉토리는 주제별로 관련 작업 로그와 공식 문서를 다시 찾기 쉽게 묶는다.

## 목적

- 날짜를 모르더라도 주제로 접근할 수 있게 한다.
- 관련 worklog, REQUIREMENTS, discussion, ADR, explainer, guide를 한곳에서 찾게 한다.

## 언제 만드는가

- 같은 주제로 여러 worklog가 누적된 경우
- 나중에 다시 찾을 가능성이 높은 경우
- 공식 문서와 구현 로그를 함께 봐야 이해되는 경우

## 언제 만들지 않는가

- 단발성 작업이고 다시 찾을 가능성이 낮은 경우
- worklog 하나만 봐도 충분한 경우
- 원본 로그를 복붙하는 것 외 의미가 없는 경우

## 권장 구조

1. 목적
2. 관련 worklog
3. 관련 문서
4. 현재 기준
5. 열어둘 질문

## 작성 원칙

- topic 문서는 탐색 문서다.
- 원본 사실과 세부 구현은 `worklog`를 기준으로 보고, 여기에는 링크와 짧은 맥락만 둔다.
- 내부 링크는 GitHub 저장소 기준 `blob/dev` 링크를 사용한다.
